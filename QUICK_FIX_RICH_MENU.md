# Rich Menu 上傳失敗 - 快速修復指南

## 🚨 問題：Rich Menu 不存在或無效

如果您看到「Rich Menu 不存在或無效」的錯誤，這表示您使用的 Rich Menu ID 不存在。

## ✅ 最簡單的解決方法

### 方法一：創建新的 Rich Menu（推薦，最簡單）

1. **在上傳工具中，點擊「創建新的 Rich Menu」按鈕**
2. **等待創建完成**（約 5-10 秒）
3. **創建成功後**：
   - Rich Menu ID 會**自動填入**上方輸入框
   - 確認輸入框中的 Rich Menu ID 已更新
4. **選擇圖片檔案**
5. **點擊「上傳圖片」按鈕**
6. **上傳成功後，點擊「設定為預設 Rich Menu」按鈕**

### 方法二：從檢查工具選擇

1. **打開 `check-rich-menu.html`**
2. **點擊「檢查 Rich Menu 列表」按鈕**
3. **查看結果**：
   - **如果有 Rich Menu 列表**：
     - 選擇一個 Rich Menu
     - 點擊「使用此 Rich Menu」按鈕
     - 點擊「打開上傳工具」按鈕
     - Rich Menu ID 會自動填入
   - **如果顯示「找到 0 個 Rich Menu」**：
     - 點擊「創建新的 Rich Menu」按鈕
     - 記下返回的 Rich Menu ID
     - 使用上傳工具上傳圖片

## 📝 完整操作流程（推薦）

### 步驟 1：創建 Rich Menu

1. 打開 `upload-rich-menu.html`
2. 點擊「創建新的 Rich Menu」按鈕
3. 等待創建完成
4. **確認 Rich Menu ID 已自動填入輸入框**

### 步驟 2：上傳圖片

1. 確認 Rich Menu ID 輸入框中有正確的 ID（應該以 `richmenu-` 開頭）
2. 點擊「選擇檔案」，選擇您的圖片
   - 圖片規格：2500 x 1686 像素
   - 格式：PNG 或 JPEG
   - 大小：< 1MB
3. 點擊「上傳圖片」按鈕
4. 等待上傳完成

### 步驟 3：設定為預設

1. 上傳成功後，會顯示「設定為預設 Rich Menu」按鈕
2. 點擊該按鈕
3. 等待設定完成

### 步驟 4：在 LINE 中查看

1. **完全關閉 LINE 應用程式**（不是只切換）
2. **等待 30 秒**
3. **重新打開 LINE**
4. **進入 LINE Bot 聊天視窗**
5. **應該會在底部看到 Rich Menu**

## ⚠️ 常見錯誤

### 錯誤 1：使用舊的 Rich Menu ID

**問題**：使用了一個不存在的 Rich Menu ID

**解決**：
- 點擊「創建新的 Rich Menu」按鈕
- 使用新創建的 Rich Menu ID

### 錯誤 2：Rich Menu ID 沒有自動填入

**問題**：創建 Rich Menu 後，ID 沒有自動填入輸入框

**解決**：
- 手動複製創建成功後顯示的 Rich Menu ID
- 貼到輸入框中

### 錯誤 3：上傳後還是看不到 Rich Menu

**問題**：已上傳圖片並設定為預設，但 LINE 中還是看不到

**解決**：
1. 確認圖片已上傳（上傳工具顯示「上傳成功」）
2. 確認已設定為預設（設定工具顯示「設定成功」）
3. **完全關閉 LINE 應用程式**（不是只切換）
4. 等待 5-10 分鐘
5. 重新打開 LINE

## 🎯 快速檢查清單

- [ ] 已創建新的 Rich Menu（或確認現有的 Rich Menu 存在）
- [ ] Rich Menu ID 已正確填入輸入框
- [ ] 已選擇符合規格的圖片（2500x1686，PNG/JPEG，< 1MB）
- [ ] 圖片已成功上傳
- [ ] 已設定為預設 Rich Menu
- [ ] 已完全關閉並重新打開 LINE
- [ ] 等待 5-10 分鐘後檢查

## 💡 重要提示

1. **必須先創建 Rich Menu，才能上傳圖片**
2. **必須先上傳圖片，才能設定為預設**
3. **設定為預設後，需要等待 5-10 分鐘讓 LINE 更新**
4. **必須完全關閉 LINE 應用程式，不是只切換**

## 🔄 如果還是失敗

如果按照以上步驟操作還是失敗，請：

1. **檢查瀏覽器 Console**（F12）查看詳細錯誤
2. **確認網路連線正常**
3. **確認 Vercel 部署已完成**（等待 1-3 分鐘）
4. **嘗試重新創建 Rich Menu**
