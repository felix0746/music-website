# 如何查看已存在的 Rich Menu

## 📍 在 LINE Official Account Manager 中查看

### 方法一：點擊「建立圖文選單」按鈕

1. **點擊綠色的「建立圖文選單」按鈕**
2. **可能會顯示已存在的 Rich Menu 列表**
3. **或者會進入創建頁面，但可以查看現有的**

### 方法二：查看已建立的 Rich Menu 列表

在 LINE Official Account Manager 中：

1. **在「圖文選單」頁面中**
2. **向下滾動或查看其他標籤**
3. **尋找「已建立的圖文選單」或「圖文選單列表」**
4. **或者點擊左側選單中的其他選項**

### 方法三：使用我們的工具（最簡單）

**不需要在 Manager 中查找**，直接使用我們的工具：

1. **打開 `check-rich-menu.html`**
2. **點擊「檢查 Rich Menu 列表」**
3. **會顯示所有已存在的 Rich Menu**

## ✅ 推薦方法：使用我們的檢查工具

由於在 Manager 中可能不容易找到已存在的 Rich Menu，建議直接使用我們的工具：

### 步驟 1：檢查 Rich Menu 列表

1. 打開 `check-rich-menu.html`
2. 點擊「檢查 Rich Menu 列表」按鈕
3. 查看返回的列表

**如果看到 Rich Menu 列表**：
- ✅ 表示 Rich Menu 存在
- ✅ 記下您要使用的 Rich Menu ID
- ✅ 可以繼續下一步

**如果顯示「找到 0 個 Rich Menu」**：
- ❌ 表示還沒有 Rich Menu
- ✅ 需要先創建 Rich Menu

### 步驟 2：確認圖片是否已上傳

由於無法在 Manager 中直接查看圖片，可以：

**方法 A：嘗試重新上傳**
1. 打開 `upload-rich-menu.html`
2. 輸入您的 Rich Menu ID
3. 選擇圖片並上傳
4. 如果上傳成功，表示圖片已上傳（或成功覆蓋）

**方法 B：檢查上傳歷史**
- 查看之前是否有「上傳成功」的訊息
- 如果有，表示圖片應該已上傳

### 步驟 3：設定為預設

1. 在 `check-rich-menu.html` 中
2. 找到您的 Rich Menu
3. 點擊「設為預設」按鈕
4. 確認設定成功

## 🎯 實際操作建議

**不需要在 Manager 中查找 Rich Menu**，請直接：

1. **使用檢查工具**（`check-rich-menu.html`）
   - 這是最簡單、最直接的方法
   - 會顯示所有 Rich Menu 及其 ID

2. **使用診斷工具**（`diagnose-rich-menu.html`）
   - 自動診斷 Rich Menu 狀態
   - 顯示問題和建議

3. **使用上傳工具**（`upload-rich-menu.html`）
   - 上傳圖片時會自動檢查 Rich Menu 是否存在
   - 如果不存在會顯示錯誤訊息

## 📝 快速檢查步驟

1. ✅ 打開 `check-rich-menu.html`
2. ✅ 點擊「檢查 Rich Menu 列表」
3. ✅ 查看返回的列表
4. ✅ 如果看到 Rich Menu，記下 ID
5. ✅ 如果沒有，點擊「創建新的 Rich Menu」

## 💡 重要提示

**LINE Official Account Manager 的 Rich Menu 頁面主要用於創建新的 Rich Menu**，查看已存在的 Rich Menu 可能不太方便。

**使用我們的工具更簡單**：
- ✅ 直接顯示所有 Rich Menu
- ✅ 顯示 Rich Menu ID、名稱、尺寸等資訊
- ✅ 可以直接操作（設定為預設、上傳圖片等）
- ✅ 不需要在 Manager 中查找
